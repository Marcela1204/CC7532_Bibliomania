# Condições dos componentes

## Pré-condições:
* registrarEmprestimo : O livro precisa estar disponível / o leitor não pode estar em debito com a biblioteca
* cadastrarLeitor : O leitor deve haver documento válido / o leitor não deve haver cadastro anterior
* cadastrarLivro :  O ISBN deve ser válido
## Pós-condição
* registrarEmprestimo : Emprestimo registrado ou emprestimo não registrado
* cadastrarLeitor : O leitor é cadastrado ou o leitor não é cadastrado
* cadastrarLivro : O livro é cadastrado no acervo ou o livro não é cadastrado no acervo
